# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview

This project is a **Heart Disease Prediction System** developed using **Python in Google Colab**. It uses the **Logistic Regression algorithm** to analyze medical data and predict whether a person is likely to have heart disease or not.

The project uses a heart disease dataset containing **1025 patient records**, 13 input features, and 1 target variable.

---

## 🚀 Features

* Data analysis and preprocessing using Pandas
* Checking missing values and dataset statistics
* Splitting data into training and testing sets
* Training a Logistic Regression machine learning model
* Evaluating model accuracy
* Predicting heart disease based on user input

---

## 🛠️ Technologies & Tools Used

* Python 🐍
* Google Colab
* NumPy
* Pandas
* Scikit-learn (Machine Learning)

---

## 📂 Dataset Information

The dataset includes medical features such as:

* Age
* Sex
* Chest pain type
* Blood pressure
* Cholesterol level
* Fasting blood sugar
* ECG results
* Maximum heart rate
* Exercise-induced angina
* ST depression
* Slope, number of vessels, and thal value

**Target:**

* 0 → No Heart Disease
* 1 → Heart Disease

---

## ⚙️ Machine Learning Model

**Algorithm Used:** Logistic Regression

### Data Split:

* Training Data: 80%
* Testing Data: 20%

---

## 📊 Model Accuracy

* Training Accuracy: **85.24%**
* Testing Accuracy: **80.48%**

---

## 🧪 Sample Prediction

Input:

(59, 1, 2, 126, 218, 1, 1, 134, 0, 2.2, 1, 1, 1)

Output:

0 → The person does not have heart disease.

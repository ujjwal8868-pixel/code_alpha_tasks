## 📌 Overview

This project implements a Machine Learning model that predicts an individual's creditworthiness using financial data and the Random Forest Classification algorithm.

## 🎯 Objective

The objective of this project is to classify whether an individual is creditworthy or not based on various financial features such as income, debt, payment history, and other financial indicators.

## 🛠 Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📊 Dataset

The project uses a synthetic credit dataset containing 1000 samples and 10 financial features, including:

* Income
* Debt
* Payment History
* Credit Age
* Number of Accounts
* Missed Payments
* Credit Limit
* Loan Amount
* Employment Years
* Monthly Expenses

## 📈 Model Performance

* Algorithm: Random Forest Classifier
* Accuracy: 94.5%
* Precision: 93.2%
* Recall: 94.3%
* F1 Score: 93.7%
* ROC-AUC Score: 94.4%
